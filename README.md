# 🐾 福間家イヌズ健康手帳

こはる・Noël・エルモ・ラルフの健康を管理するWebアプリです。

## URL

https://arisaf1201-doggo.github.io/doggo-vactination-app/dog-health-app.html

## 機能

### 👤 プロフィール
- 誕生日・家族になった日を記録
- 年齢・記念年数をヶ月単位で自動計算表示
- 各犬のページとみんなの状況一覧に表示

### ⚠️ アラート（犬ごとに表示）
- フィラリア当月未投与の場合に表示
- ワクチン次回予定日が60日以内 or 未記録の場合に表示
- 次回予定日・残り日数を表示

### 💉 ワクチン管理
- **狂犬病・混合ワクチン**の2種類
- 接種履歴を複数記録（過去の記録もすべて残せる）
- 混合ワクチンは接種した種類も記録可能
- 次の接種記録がある場合は「次回」表示を非表示
- 最新記録から次回接種予定日を自動計算

### 💊 フィラリア予防管理
- 月ごとに投与日を記録（タップでモーダル入力）
- **「不要」選択肢あり**（投薬不要の月は算定対象外）
- 前回投与から1ヶ月後の期限を表示
- 今年の投与率をプログレスバーで表示（不要月は除外）

### ⚖️ 体重管理
- 日付と体重を記録
- 折れ線グラフで推移を表示（2件以上から）
- 最新体重・前回比・記録数を表示
- 記録履歴の削除も可能

### 📋 みんなの状況（一覧）
- 4匹まとめてプロフィール・ワクチン・フィラリア・体重の状況を確認

## データ
- **Firebase Firestore**に保存（リアルタイム同期）
- PC・スマホどちらから入力しても即時反映
- URLを共有すれば家族も閲覧・編集可能

## ブランチ
- `main` : 本番
- `develop` : 開発用

## スマホでの使い方
Safariで開いて「ホーム画面に追加」するとアプリのように使えます。

## 開発メモ
- Firebase project: `doggo-health`
- Firestore key: `wanko/records`
- フィラリア「不要」は `"skip"` で保存
- ワクチンデータ形式: `{date: "YYYY-MM-DD", type: "種類"}`
- プロフィールデータ形式: `{birthday: "YYYY-MM-DD", anniversary: "YYYY-MM-DD"}`
