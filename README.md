# 🐾 わんこ健康手帳

こはる・Noël・エルモ・ラルフのワクチンとフィラリア予防を管理するWebアプリです。

## URL

https://arisaf1201-doggo.github.io/doggo-vactination-app/dog-health-app.html

## 機能

### ⚠️ アラート（犬ごとに表示）
- フィラリア当月未投与の場合に表示
- ワクチン次回予定日が60日以内 or 未記録の場合に表示
- 次回予定日・残り日数を表示

### 💉 ワクチン管理
- **狂犬病・混合ワクチン**の2種類
- 接種履歴を複数記録（過去の記録もすべて残せる）
- 混合ワクチンは接種した種類も記録可能
- 次の接種記録がある場合は「次回」表示を非表示
- 最新記録から次回接種予定日を自動計算

### 💊 フィラリア予防管理
- 月ごとに投与日を記録（タップでモーダル入力）
- 前回投与から1ヶ月後の期限を表示
- 今年の投与率をプログレスバーで表示

### 📋 全犬の状況一覧
- 4匹まとめてワクチン・フィラリアの状況を確認

## データについて

- データはブラウザのlocalStorageに保存されます
- **シークレットブラウザでは保存されません**
- 他の端末とはデータ共有されません

## スマホでの使い方

Safariで開いて「ホーム画面に追加」するとアプリのように使えます。

- localStorage key: `wanko4`
